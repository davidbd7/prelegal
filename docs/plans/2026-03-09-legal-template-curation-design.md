# KAN-2: Legal Template Dataset Curation

## Summary

One-time data curation task to populate the Prelegal project with CommonPaper legal agreement templates sourced from https://github.com/CommonPaper under CC BY 4.0 license.

## Source

12 markdown template files across 11 CommonPaper GitHub repos:
Mutual-NDA (2 files), CSA, SLA, Design-Partner-Agreement, PSA, DPA, Partnership-Agreement, Software-License-Agreement, Pilot-Agreement, BAA, AI-Addendum.

## Deliverables

1. `templates/` directory containing all 12 markdown files with original filenames
2. `templates/LICENSE.txt` with CC BY 4.0 attribution notice
3. `catalog.json` in project root with name, description, and filename for each template
